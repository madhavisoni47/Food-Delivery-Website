# Food-Delivery-Website
A responsive Food Delivery Website built using HTML, CSS, and JavaScript.
This project simulates an online food ordering platform where users can browse food items, add them to cart, and view total pricing dynamically.


**#Features**
Responsive design (Mobile + Desktop)
Dynamic product rendering from JSON file
Add to Cart functionality
Increase/Decrease item quantity
Real-time total price calculation
Clean and modern UI

 **Technologies Used**

HTML5
CSS3
JavaScript (ES6)
Fetch API
